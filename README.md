# Python Student Record Saver

## Description

This project demonstrates Python file handling by storing student names and marks in a text file. It allows users to save, read, and display student records while practicing file operations.

## Features

- Save student records to a text file
- Read records from a file
- Display saved data
- Append new student records
- Practice file handling

## Technologies Used

- Python 3

## Concepts Practiced

- File Handling
- Read Mode (`r`)
- Write Mode (`w`)
- Append Mode (`a`)
- User Input
- Loops
- Variables

## Future Improvements

- Store records in CSV format
- Search for student records
- Edit existing records
- Delete student records
